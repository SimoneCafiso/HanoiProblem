# Hanoi's tower algorithm

Just the hanoi algorithm
## Description

Simple hanoi's tower algorithm made up in c++!


Object of the problem is to move all the disks over to Tower 3.
But you cannot place a larger disk onto a smaller disk.

## Getting Started

### Executing program

* How to run the program with cmake
```
$ mkdir build

$ cd build

$ cmake ..

$ cmake --build .

```


* How to run the program with clang
```
g++ -o hanoi main.cpp
```

### Sample Input
```
6
```
